Claude Code 設定の引き継ぎ

既にClaude Codeを使っている方向け。
RIOのスキル・設定をあなたのClaude Codeに追加します。
OVERVIEW
何をするの？
RIOが作った便利な設定（スキル、安全ルール、自動機能など）を、あなたの既存のClaude Codeに追加します。
今の設定は消えません。上書きではなく、足し算です。
注意
この作業はターミナル（Claude Codeの画面）で行います。
不安な場合は、Claude Codeに「この引き継ぎガイドの通りに設定して」と言えばやってくれます。
STEP 1
このZIPの中にある設定ファイルを確認
1ZIPを解凍する
ダウンロードしたZIPをダブルクリックで解凍。
「claude-starter-kit」というフォルダが出てきます。
2隠しフォルダがあることを知っておく
フォルダの中に、Finderでは見えない .claude というフォルダが入っています。
ここにスキルや設定が全部入っています。見えないけど、ちゃんとあります。
隠しフォルダの見方（気になる人だけ）
Finderで Command + Shift + .（ドット）を押すと、隠しファイルが薄く表示されます。もう一度押すと元に戻ります。
STEP 2
Claude Codeに引き継ぎをお願いする
一番簡単な方法は、Claude Code自身にやってもらうことです。
Claude Codeを開いて、以下をそのままコピペしてください。
1Claude Codeで以下を実行
ターミナルでClaude Codeを起動し、以下を貼り付けてください。
あなた > 以下のフォルダにRIOからもらった設定一式があります。 私の既存の設定を壊さないように、追加・マージしてください。 場所: ~/Downloads/claude-starter-kit/.claude/ やってほしいこと: 1. スキル（skills/フォルダ内）を全て ~/.claude/skills/ にコピー 2. CLAUDE.mdの内容を、私の既存の ~/.claude/CLAUDE.md に追記 （既にある内容は消さないで、末尾に追加） 3. hooks/ と scripts/ を ~/.claude/ にコピー 4. docs/ を ~/.claude/ にコピー 5. 完了したら何を追加したか教えて
ポイント
Claude Codeはファイル操作が得意なので、「壊さないでね」と伝えれば安全にやってくれます。
万が一心配なら「作業前にバックアップを取って」と一言添えましょう。
STEP 3
確認する
1スキルが入ったか確認
Claude Codeに以下を聞いてみてください。
あなた > 今使えるスキルの一覧を教えて
2追加されたスキルの例
以下のようなスキルが使えるようになっていればOKです。

・company-research — 企業調査レポートを自動生成
・deepdive — タスクの深掘り調査
・industry-report-html — 業界レポートをHTMLに変換
・x-research — X（Twitter）でリアルタイム情報を検索
・copywriting — マーケティングコピーを作成
・その他多数
OPTIONAL
自分でやりたい場合（上級者向け）
Claude Codeに任せず自分でやりたい場合のコマンドです。
# 1. スキルをコピー（既存は上書きしない） cp -rn ~/Downloads/claude-starter-kit/.claude/skills/* ~/.claude/skills/ # 2. CLAUDE.mdを追記 cat ~/Downloads/claude-starter-kit/.claude/CLAUDE.md >> ~/.claude/CLAUDE.md # 3. フック・スクリプト・docsをコピー cp -rn ~/Downloads/claude-starter-kit/.claude/hooks/* ~/.claude/hooks/ cp -rn ~/Downloads/claude-starter-kit/.claude/scripts/* ~/.claude/scripts/ cp -rn ~/Downloads/claude-starter-kit/.claude/docs/* ~/.claude/docs/
注意
ダウンロード先がDownloadsフォルダでない場合は、パスを自分の解凍先に置き換えてください。
引き継ぎ完了
これでRIOのスキルと設定が使えるようになりました。
